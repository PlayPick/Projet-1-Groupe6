========================================================================================================================================
   Dans cette branche se trouve le dossier du programme final, c'est à dire la combinaison des programmes de tous les autres modules 
========================================================================================================================================

Pour executer le programme il faut : 
- Telecharger le dossier 
- Ouvrir puis compiler/téléverser le programme arduino si ce n'est deja fait
- Ouvrir le programme processing et l'exécuter pendant l'utilisation de l'Arduino
- Créer le fichier exécutable en C si ce n'est déjà fait, et l'exécuter. 
- Fermer le processing quand on veut arrêter de prendre des valeurs
- Choisir dans le menu du programme en C l'action à réaliser
